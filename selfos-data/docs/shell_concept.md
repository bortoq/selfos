# Self OS Shell — Концепция

## Идея

Self OS Shell — это **единая командная поверхность**, через которую пользователь может управлять всеми своими цифровыми инструментами без необходимости переключаться между приложениями.

## Основные принципы

1. **Единая точка входа**
   - CLI
   - Веб-интерфейс
   - Voice / Chat интерфейс

2. **Контекстность**
   - Система понимает текущий контекст пользователя (что он делает, где находится, какое время)

3. **Прогрессивное делегирование**
   - Пользователь может постепенно передавать всё больше контроля Self OS

## Предлагаемые команды (MVP)

| Команда                    | Описание                              | Пример |
|---------------------------|---------------------------------------|--------|
| `selfos note <text>`      | Быстрая заметка с делегированием      | `selfos note "Buy milk"` |
| `selfos task <text>`      | Создать задачу                        | `selfos task "Finish report"` |
| `selfos status`           | Показать текущий статус дня           | `selfos status` |
| `selfos suggest`          | Получить умные предложения            | `selfos suggest` |
| `selfos delegate <action>`| Включить/выключить делегирование      | `selfos delegate photo_classification` |
